# Dotfiles

My dotfiles for settings up my machine.

## Prerequisites

- [ZSH](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH)
- [stow](https://www.gnu.org/software/stow/)
- [Oh-My-Zsh](https://ohmyz.sh/#install)
- [Powerlevel 10k](https://github.com/romkatv/powerlevel10k?tab=readme-ov-file#oh-my-zsh)
- [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions/blob/master/INSTALL.md#oh-my-zsh)
- [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting/blob/master/INSTALL.md#oh-my-zsh)

## Getting Started

### Supported OS's

- Arch Linux

```bash
URL="https://raw.githubusercontent.com/jgttech/dotfiles/refs/heads/main/install" DIR="dotfiles"; wget -qO- $URL | bash /dev/stdin $DIR;
```
