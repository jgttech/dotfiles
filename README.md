# Dotfiles

My personal dotfiles.

## Prerequisites

- [ZSH](https://github.com/ohmyzsh/ohmyzsh/wiki/Installing-ZSH)
  - Need to install and configure a setup before installing the dotfiles.
- [Oh My ZSH](https://ohmyz.sh/#install) (Optional)
  - Could set this up, if desired.
- [Powerlevel10k](https://github.com/romkatv/powerlevel10k) (Optional)
  - Could set this up, if desired.

## Getting Started

> Arch Linux

```bash
URL="https://raw.githubusercontent.com/jgttech/dotfiles/refs/heads/main/.bin/install" DIR="dotfiles"; INSTALL="pacman-install"; wget -qO- $URL | bash /dev/stdin $DIR $INSTALL;
```
